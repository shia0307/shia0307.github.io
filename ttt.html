Tweak the Tweet

Tweak the Tweet design specification implementation.

Duration

1 Week, Course Project, Feb, 2014

Team Member

Hadiza Ismaila, Albert Lui, Punchet Sangnil, David Tianxiao Yang

Applied Skills

Design Implementation, Coding, UI Design

Deliverables

Blog Post, Mobile Prototype, Arduino Prototype

Project Summary

Tweak the Tweet is a mobile application that helps build tweets that report and map information regarding a disaster, crisis or emergency. In this one week assignment, we are challenged to implement this application which a design specificatino has already been produced.

This is for a system that is part of Human Centered Design and Engineering Professor Kate Starbird's work in crisis research in University of Washington. She studies the "use of social media during crisis events", specifically looking at how the converging audience can contribute.

Mobile Prototype

The video demo above is our working prototype of Tweak the Tweet. We used the software called App Inventor to implement it. App Inventor is a blocks-based programming tool that allows everyone, even novices, to start programming and build fully functional apps for Android devices.

Design Revised

Even with the detailed design specification, we still encountered a lot of difficulties. Due to the constraints of using App Inventor, the original interface could not be properly layout. We dicided to redesign the interface to suit better with the tools we use.

Final Prototype

Here is final prototype we designed. Because the application is used in emergency environment, we automated a lot of steps in the flow to minimize the user input. The final prototype is a functioning prototyep with Google Maps and Twitter API.
