Project Summary

I wanted to have a personal portfolio website that can showcase my work. I have investigated many options, finally I decided to learn HTML and CSS to code my own website.

This project not only showcase my final design of my website, but also documented my learning process - from zero coding experience to a final working website.

Portfolio Options

Before I dicided to use HTML+CSS to code my own website, I did some research on what's available at the moment and listed their pros and cons. As a designer, I valued creativities and uniqueness, and I saw this a great opportunity for me to learn new skills. Finally I decided to moved on with coding my own website.

Blog posting services do not offer many option for layouts and they are not the best tools for picture display and product showcase.

Portfolio website service has less flexibility, and they all look the same.

Coding need a lot of time commitment and patience.

Learning Process

It was a long and winding journey. I started my learning process in May 2014, and it took me about two months to finish learning basic HTML+CSS skill. I then started to build my website and configured my domain and hosting.

In the whole process of building a website, I found configuring the hosting and domain on domain provider is the hardest part of all. They don't speak the human language on the website, I had to search on every technical term they use. It was a very frustraing experience.

This book helped me tremendously, it taught HTML and CSS in an easy way, and it's visually appealing wih good layouts and structure, which is very important to designers.

I also used bootstrap to code, with it's framework, it made my life so much more easier.
